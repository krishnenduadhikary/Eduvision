1. Introduction to Fuzzy Logic

Fuzzy logic, introduced by Lotfi A. Zadeh in the 1960s, is an approach to computing based on "degrees of truth" rather than the usual "true or false" (1 or 0) Boolean logic on which the modern computer is based.

  • It is a superset of Boolean (conventional) logic that handles the concept of partial truth – truth values between "completely true" and "completely false".
  • Fuzzy logic is multivalued. It deals with degrees of membership and degrees of truth.
  • It uses the continuum of logical values between 0 (completely false) and 1 (completely true).

Example: Consider a 100 ml glass containing 30 ml of water.

  • In Boolean logic, we might say the glass is "not full" (true) and "not empty" (true), or perhaps "half empty" (false) if we only have strict categories.
  • In fuzzy logic, one might define the glass as being 0.7 empty and 0.3 full, assigning degrees of membership to these linguistic concepts.

Illustrative difference between Boolean (crisp) and Fuzzy interpretation for a glass of water.

Fuzzy logic is particularly useful for systems that are difficult to model precisely or involve human-like reasoning and ambiguity. Its key advantages include:

  • Based on intuition and judgment, allowing for the incorporation of expert knowledge in a more natural way.
  • Often, there is no need for a complex mathematical model of the system.
  • Provides a smooth transition between members and non-members of a set, avoiding abrupt changes.
  • Systems can be relatively simple, fast, and adaptive.
  • Fuzzy systems are generally less sensitive to system fluctuations and noise in input data.
  • Can implement design objectives that are difficult to express mathematically, using linguistic or descriptive rules (e.g., "if temperature is very high and humidity is slightly high, then increase fan speed moderately").

2. Fuzzy Sets vs. Crisp Sets

Crisp Logic (Boolean Logic)

  • A proposition can be True or False only. There are no intermediate values.
  • Examples:
    • Bob is a student (True / False).
    • Smoking is healthy (False).
  • The degree of truth is either 0 or 1.

Fuzzy Logic

  • The degree of truth is a value between 0 and 1 (inclusive).
  • Examples:
    • William is young (Truth value = 0.3, meaning "somewhat young").
    • Ariel is smart (Truth value = 0.9, meaning "very smart").

Crisp Sets

Classical sets are also called 'crisp' sets. In a crisp set, an element x from the universe X is either a member of set A or it is not. There is a sharp, unambiguous boundary.

Example: The set of "Tall People" defined as anyone 1.8m or taller. A person 1.79m is NOT tall; a person 1.80m IS tall.

Non-Crisp Sets (Fuzzy Sets)

In fuzzy set theory, this clear-cut membership is generalized. It is not necessary that an element x is a full member of the set or not a member at all. It can be a partial member of the set, described by a degree of membership.

Example: For the fuzzy set "Tall People," a person 1.79m might have a membership degree of 0.8 (quite tall), while a person 1.80m has a degree of 0.9 (very tall), and someone 1.70m might have 0.2 (slightly tall).

Representation of "Tall" using Crisp vs. Non-Crisp (Fuzzy) Logic.

If students with 1.8m height are qualified for a basketball team:

  • In crisp set theory, only students with height ≥ 1.8m are considered tall. A student 1.79m tall is not qualified.
  • In non-crisp (fuzzy) set theory, a student with height 1.78m might belong to both "tall" and "not tall" sets with particular degrees of membership (e.g., 0.7 tall, 0.3 not tall).

3. Crisp Set Theory (Foundation)

Understanding classical (crisp) set theory is fundamental before diving deeper into fuzzy sets, as fuzzy set theory generalizes many concepts from crisp sets.

Universe of Discourse (E or U)

The universe of discourse, or universal set, is the set which, with reference to a particular context, contains all possible elements having the same characteristics and from which sets can be formed. Denoted by E or U.

Examples:

  • The universal set of all numbers in Euclidean space.
  • The universal set of all students in a university.

Set

A set is a well-defined collection of distinct objects. "Well-defined" means that an object either belongs to the set or does not belong to the set (crispness).

Examples:

  • A = {Gandhi, Bose, Nehru}
  • B = {Swan, Peacock, Dove}
  • A = {x | x is an odd number}
  • B = {y | y > 0 and y mod 5 = 0}

Membership

An element x is said to be a member of a set A if x belongs to the set A. This is indicated by '∈' (belongs to) and '∉' (does not belong to).

Example: Given A = {4, 5, 6, 7, 8, 10}, for x = 3 and y = 4, we have x ∉ A and y ∈ A.

Cardinality

The number of elements in a set is called its cardinality, denoted as n(A) or |A|.

Example: If A = {4, 5, 6, 7}, then |A| = 4.

Other Definitions

  • Family of sets: A set whose members are sets themselves. E.g., A = {{1, 3, 5}, {2, 4, 6}, {5, 10}}.
  • Null Set/Empty Set (∅ or {}): A set that has no members. |∅| = 0. E.g., The set of all prime ministers who are below 15 years of age.
  • Singleton Set: A set with a single element. It has cardinality of 1. E.g., If A = {a}, then |A| = 1.
  • Subset (A ⊆ B): Set A is a subset of B if every element of A is also in B. (A ⊂ B for proper subset).
  • Superset (A ⊇ B): Set A is a superset of B if every element of B is also in A. (A ⊃ B for proper superset).
  • Power Set (P(A)): The set of all possible subsets of set A, including the null set and A itself. Cardinality |P(A)| = 2|A|.
    Example: If A = {3, 4}, P(A) = {∅, {3}, {4}, {3, 4}}. |P(A)| = 22 = 4.

Fig. Venn diagram of a set A within universal set E.

Union (A ∪ B)

The set of all elements that belong to A or to B or to both. A ∪ B = {x | x ∈ A or x ∈ B}

Fig. Venn Diagram for A ∪ B

Intersection (A ∩ B)

The set of all elements that belong to both A and B. A ∩ B = {x | x ∈ A and x ∈ B}

Fig. Venn Diagram for A ∩ B

Complement (Ac or A')

The set of all elements in the universal set E that are not in A. Ac = {x | x ∈ E and x ∉ A}

Fig. Venn Diagram for Ac

Difference (A - B)

The set of all elements that are in A but not in B. A - B = {x | x ∈ A and x ∉ B} (Also A ∩ Bc)

Fig. Venn Diagram for A - B

  • Commutativity:
    • A ∪ B = B ∪ A
    • A ∩ B = B ∩ A
  • Associativity:
    • (A ∪ B) ∪ C = A ∪ (B ∪ C)
    • (A ∩ B) ∩ C = A ∩ (B ∩ C)
  • Distributivity:
    • A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
    • A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
  • Idempotence:
    • A ∪ A = A
    • A ∩ A = A
  • Identity:
    • A ∪ ∅ = A
    • A ∩ E = A
    • A ∩ ∅ = ∅
    • A ∪ E = E
  • Law of Absorption:
    • A ∪ (A ∩ B) = A
    • A ∩ (A ∪ B) = A
  • Transitivity: If A ⊆ B and B ⊆ C, then A ⊆ C
  • Involution: (Ac)c = A
  • Law of the Excluded Middle: A ∪ Ac = E
  • Law of Contradiction: A ∩ Ac = ∅
  • De Morgan's Laws:
    • (A ∪ B)c = Ac ∩ Bc
    • (A ∩ B)c = Ac ∪ Bc

4. Crisp Relations and Operations

A crisp relation represents the presence or absence of association, interaction, or interconnectedness between two or more elements or sets.

Cartesian Product (X × Y)

Let X and Y be two crisp sets. The Cartesian product of X and Y is a set of all ordered pairs (x, y) such that x ∈ X and y ∈ Y. It is represented by X × Y.

X × Y = {(x, y) | x ∈ X, y ∈ Y}

Example: Let X = {8, 5}, Y = {2, 4, 7}.
X × Y = {(8, 2), (8, 4), (8, 7), (5, 2), (5, 4), (5, 7)}

Binary Relation (R)

A binary relation R among crisp sets X and Y is a subset of the Cartesian product X × Y (i.e., R ⊆ X × Y). It represents the strength of association between elements of the two sets.

Example 1: Let M be the set of males {Mike, Charles, James} and F be the set of female {Gauri, Rita, Mira}. The relation C "couple on M × F" could be:
C = {(Charles, Gauri), (Mike, Mira), (James, Rita)}

Example 2: Relation 'Less than' (<) from set of natural numbers N to N: { (0,1), (0,2)...(1,2)...}

Characteristic Function for a Binary Relation

A binary relation R can be defined by a characteristic function that assigns a value 1 to every pair (x,y) that belongs to the relation and 0 to every pair that does not belong to the relation.

Example: Let X = {India, UK, USA, Germany}, Y = {dollar, rupee, mark, pound, Euro}. Let R be a crisp relation "associates a country with a currency".
R(India, rupee) = 1, R(UK, pound) = 1, etc. Other relations like R(India, euro) = 0.

This can be represented as a matrix:

Two-dimensional membership matrix for country-currency relation.

Representations of Binary Relations

Binary relations can be represented by:

  • Coordinate diagrams (for relations like x² + y² = 4)
  • Directed graphs (binary relation from A to B)
  • Relation matrices

Examples of binary relation representations.

Function-theoretic operations for two crisp relations (R, S) are defined as follows (using characteristic functions χ):

  1. Union (R ∪ S): χR∪S(x, y) = max[χR(x, y), χS(x, y)]
  2. Intersection (R ∩ S): χR∩S(x, y) = min[χR(x, y), χS(x, y)]
  3. Complement (R̄): χR̄(x, y) = 1 - χR(x, y)
  4. Containment (R ⊂ S): χR(x, y) ≤ χS(x, y)
  5. Identity (∅ → ∅R and X → ER)

Given R is a relation on X, Y and S is another relation on Y, Z. Then R ∘ S is called a composition of relation on X and Z.

R ∘ S = {(x, z) | (x, y) ∈ R and (y, z) ∈ S for ∀y ∈ Y}

Max-Min Composition

Given two relation matrices R and S, the max-min composition is defined as T = R ∘ S;

T(x, z) = maxy{min{R(x, y), S(y, z)}} for ∀y ∈ Y

Example: Given X = {1,3,5}; Y = {1,3,5}; R = {(x,y)|y = x+2}; S = {(x,y)|x < y}

Example of Max-Min Composition of Crisp Relations.

Properties of Crisp Relations

The properties of crisp sets (Commutativity, Associativity, Distributivity, Involution, Idempotency, DeMorgan’s Law, Excluded Middle Laws) hold good for crisp relations as well.

5. Fuzzy Set Theory

L. Zadeh introduced fuzzy sets in the mid-60s to mathematically represent uncertainty and vagueness. It provides formalized tools for dealing with inherent imprecision in many problems.

Universe of Discourse (U)

A collection of all items that can come into consideration in a particular context.

Fuzzy Set (A)

Let U be a non-empty set called a universe of discourse. A fuzzy set A in U is defined as a set of ordered pairs:

A = {(x, μA(x)) | x ∈ U}

where μA(x) is the membership function (MF) and μA(x) is the degree of membership of element x in the fuzzy set A, for x ∈ U. The value μA(x) is in the range [0, 1].

Membership Function (μ)

A membership function μ defines the degree of belongingness to a fuzzy set between 0 and 1. Fuzzy set members are taken from a universe U.

Note: μA(x) maps each element of X onto a membership grade (or membership value) between 0 and 1 (both inclusive).

Examples of Fuzzy Sets

  • Ages of People: U = {1, 10, 20, ..., 100}. Fuzzy sets like Young, Middle_aged, Old.
    • Young = {(1, 1), (10, 0.9), (20, 0.8), (30, 0.6), (40, 0.4), (50, 0.2), (60, 0)}
    • Middle_aged = {(20, 0.2), (30, 0.4), (40, 0.7), (50, 0.9)}
    • Old = {(30, 0.1), (40, 0.3), (50, 0.5), (60, 0.8), (70, 1), (80, 1), (90, 1)}
  • Discrete Non-ordered Universe: U = {Delhi, Bombay, Bangalore} (cities).
    Fuzzy set A = "desirable city to live in": A = {(Delhi, 0.9), (Bombay, 0.8), (Bangalore, 0.6)}
  • Discrete Ordered Universe: U = {0, 2, 3, 4, 5, 6} (number of children).
    Fuzzy set A = "sensible number of children": A = {(0, 0.1), (1, 0.5), (2, 1), (3, 0.8), (4, 0.3), (5, 0.1)}
  • Continuous Universe: Let X be a set of possible ages for human.
    Fuzzy set A = "about 50-years old" may be expressed as: μA(x) = 1 / (1 + ((x - 50) / 10)4)

6. Membership Functions (MFs)

Membership functions are crucial in fuzzy logic as they define how each point in the input space is mapped to a membership value (or degree of membership) between 0 and 1.

Triangular MFs

Specified by three parameters {a, b, c}.

triangle(x; a, b, c) = 0, if x ≤ a (x-a)/(b-a), if a < x ≤ b (c-x)/(c-b), if b < x < c 0, if x ≥ c

Triangular Membership Function

Trapezoidal MFs

Specified by four parameters {a, b, c, d}.

trapezoid(x; a, b, c, d) = 0, if x ≤ a (x-a)/(b-a), if a < x ≤ b 1, if b < x ≤ c (d-x)/(d-c), if c < x < d 0, if x ≥ d

Trapezoidal Membership Function

Gaussian MFs

Specified by two parameters {c, σ} (center and width).

gaussian(x; c, σ) = e-((x-c)² / (2σ²))

Gaussian Membership Function

Generalized Bell MFs (Cauchy MF)

Specified by three parameters {a, b, c}.

bell(x; a, b, c) = 1 / (1 + |(x-c)/a|2b)

Generalized Bell Membership Function

Different shapes of Bell MF by varying parameters.

Sigmoidal MFs

Parameters {a, c} (slope at c, crossover point c).

sigmoid(x; a, c) = 1 / (1 + e-a(x-c))

Sigmoidal Membership Function

Example: Grading system for a course.

Fuzzy implementation of a grading system using sigmoidal MFs.

  • Support: The set of all points x ∈ X such that μA(x) > 0.

    Support of a Membership Function

  • Core: The set of all points x ∈ X such that μA(x) = 1.

    Core of a Membership Function

  • Normality: A fuzzy set A is normal if its core is non-empty (i.e., there is at least one point x where μA(x) = 1).

    Example of a non-normal fuzzy set (Normality = FALSE)

  • Crossover Point: A point x ∈ X at which μA(x) = 0.5.

    Crossover Point of a Membership Function

  • Fuzzy Singleton: A fuzzy set whose support is a single point in X with μA(x) = 1.

    Fuzzy Singleton

  • α-cut (Alpha-cut): The α-cut of a fuzzy set A (denoted Aα) is a crisp set defined by Aα = {x | μA(x) ≥ α}.
    Strong α-cut: Aα' = {x | μA(x) > α}.
  • Convexity: A fuzzy set A is convex if and only if for any x1, x2 ∈ X and any λ ∈ [0, 1], μA(λx1 + (1-λ)x2) ≥ min(μA(x1), μA(x2)). Visually, the MF has no "dips".

    Convex and Non-convex Membership Functions

  • Bandwidth: For a normal and convex fuzzy set, the bandwidth is the distance between the two unique crossover points.
  • Symmetry: A fuzzy set A is symmetric if its MF around a certain point x = c is μA(x + c) = μA(x - c) for all x ∈ X.

    Symmetric Membership Function

  • Open Left, Open Right, Closed: Describes the behavior of the MF as x approaches -∞ and +∞.

    Open Left, Closed, and Open Right Membership Functions

7. Operations on Fuzzy Sets

Similar to crisp sets, operations like union, intersection, and complement are defined for fuzzy sets, but they use the membership values.

Union (A ∪ B) or (A OR B)

The membership function of the union is typically defined using the maximum operator:

μA∪B(x) = max(μA(x), μB(x))

Sometimes denoted as μA(x) ∨ μB(x).

Example of Fuzzy Union (max operator)

Intersection (A ∩ B) or (A AND B)

The membership function of the intersection is typically defined using the minimum operator:

μA∩B(x) = min(μA(x), μB(x))

Sometimes denoted as μA(x) ∧ μB(x).

Example of Fuzzy Intersection (min operator)

Complement (Ac or NOT A)

The membership function of the complement is typically defined as:

μAc(x) = 1 - μA(x)

Example of Fuzzy Complement

  • Algebraic Product (A ⋅ B): μA⋅B(x) = μA(x) ⋅ μB(x)
  • Scalar Product (α × A): μαA(x) = α ⋅ μA(x) (where α is a scalar, 0 ≤ α ≤ 1)
  • Sum (A + B): μA+B(x) = μA(x) + μB(x) - μA(x) ⋅ μB(x) (Probabilistic Sum)
  • Difference (A - B): Typically μA-B(x) = μA∩Bc(x) = min(μA(x), 1 - μB(x))
  • Disjunctive Sum (A ⊕ B): μA⊕B(x) = (Ac ∩ B) ∪ (A ∩ Bc)
  • Bounded Sum (|A(+)B|): μA(+)B(x) = min(1, μA(x) + μB(x))
  • Bounded Difference (|A(-)B|): μA(-)B(x) = max(0, μA(x) + μB(x) - 1)
  • Equality (A = B): Two fuzzy sets A and B are equal if μA(x) = μB(x) for all x ∈ U.

Power of a Fuzzy Set (Aα)

The membership function is given by: μAα(x) = (μA(x))α

  • If α > 1, it is called Concentration (e.g., "very"). This operation reduces membership values, making the fuzzy set more specific.
  • If α < 1 (and α> 0), it is called Dilation (e.g., "somewhat" or "more or less"). This operation increases membership values, making the fuzzy set broader.

Cartesian Product (A × B)

The Cartesian product of two fuzzy sets A and B is denoted by A × B. Its membership function is typically defined as:

μA×B(x, y) = min(μA(x), μB(y))

Example:
A(x) = {(x1, 0.2), (x2, 0.3), (x3, 0.5), (x4, 0.6)}
B(y) = {(y1, 0.8), (y2, 0.6), (y3, 0.3)}
The resulting matrix for A × B would have μA×B(x1, y1) = min(0.2, 0.8) = 0.2, etc.

Example of Fuzzy Cartesian Product Matrix.

Many properties from classical set theory also hold for fuzzy set theory (using standard min/max operators):

  • Commutativity: A ∪ B = B ∪ A; A ∩ B = B ∩ A
  • Associativity: (A ∪ B) ∪ C = A ∪ (B ∪ C); (A ∩ B) ∩ C = A ∩ (B ∩ C)
  • Distributive Laws:
    • A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
    • A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
  • De Morgan's Laws: (A ∪ B)c = Ac ∩ Bc; (A ∩ B)c = Ac ∪ Bc
  • Absorption: A ∩ (A ∪ B) = A; A ∪ (A ∩ B) = A
  • Idempotency: A ∪ A = A; A ∩ A = A

However, the Law of the Excluded Middle and the Law of Contradiction do not generally hold for fuzzy sets:

  • A ∪ Ac ≠ U (Universal set, where all memberships are 1)
    Example: If μA(x) = 0.6, then μAc(x) = 0.4. max(0.6, 0.4) = 0.6 ≠ 1.
  • A ∩ Ac ≠ ∅ (Empty set, where all memberships are 0)
    Example: If μA(x) = 0.6, then μAc(x) = 0.4. min(0.6, 0.4) = 0.4 ≠ 0.

8. Fuzzy Relations

Fuzzy relations extend crisp relations by allowing degrees of relationship between elements of sets.

Fuzzy Cartesian Product (A × B)

Suppose A and B are two fuzzy sets defined on universal sets U and V, respectively. The Cartesian product A × B is a fuzzy set in U × V with the membership function typically defined as:

μA×B(x, y) = min(μA(x), μB(y)) for x ∈ U, y ∈ V

Example: A = {(3, 0.5), (5, 1), (7, 0.6)}, B = {(3, 1), (5, 0.6)} in universal set N.
The fuzzy Cartesian product includes pairs like ((3,3), min(0.5,1)=0.5), ((3,5), min(0.5,0.6)=0.5), etc.

Fuzzy Relation (R)

A fuzzy relation R is a fuzzy subset of A × B. The elements of the relation are defined as ordered pairs of two items of fuzzy sets grouped with its membership grade in the range [0, 1].

μR(x, y) = min(μA(x), μB(y)) (Often, a fuzzy relation is directly defined by its MF μR(x,y) over U × V)

Fuzzy relation represents the strength of association between elements of the two sets. It can be defined on both crisp and fuzzy sets.

Example: Fuzzy relation R on A × B can be represented as a matrix. Let A = {(x1, 0.2), (x2, 0.4), (x3, 0.8)}; B = {(y1, 0.3), (y2, 0.7)} If μR(x,y) = min(μA(x),μB(y)), then:

Fuzzy Relation Matrix Example

Since fuzzy relations from A to B are fuzzy sets in A × B, the operations on fuzzy sets can be extended to fuzzy relations. Let R and S be fuzzy relations.

  • Union (R ∪ S): μR∪S(x, y) = max(μR(x, y), μS(x, y))
  • Intersection (R ∩ S): μR∩S(x, y) = min(μR(x, y), μS(x, y))
  • Complement (R̄): μR̄(x, y) = 1 - μR(x, y)
  • Containment (R ⊂ S): μR(x, y) ≤ μS(x, y) for all (x,y)
  • Identity Relation (I): μI(x, y) = 1 if x = y, and 0 if x ≠ y (on a single set U)
  • Null Relation (Z): μZ(x, y) = 0 for all x, y
  • Universal Relation (U): μU(x, y) = 1 for all x, y

Example of Intersection and Union of Fuzzy Relation Matrices

Consider two binary fuzzy relations R(A, B) and S(B, C). The composition of R and S, denoted R ∘ S, is a fuzzy relation on A × C.

Max-Min Composition

This is the most common type. The membership function is:

μR∘S(x, z) = maxy∈B {min(μR(x, y), μS(y, z))}

Max-Product Composition

The membership function is:

μR∘S(x, z) = maxy∈B {μR(x, y) ⋅ μS(y, z)}

Example of Max-Min Composition of Fuzzy Relations.

Properties of Fuzzy Sets (often discussed with relations)

  • Cardinality (Scalar Cardinality): The sum of the membership values of all elements in the fuzzy set A. |A| = Σx∈U μA(x)
  • Height (h(A)): The highest membership degree in a fuzzy set A. h(A) = maxx∈U μA(x)
  • Normalization (NORM): A fuzzy set is normalized if its height is 1. If not, it can be normalized by dividing each membership degree by its height: μNormalized A(x) = μA(x) / h(A)

Reshaping of Fuzzy Operators (Linguistic Hedges)

These operators modify existing fuzzy sets.

  • Dilation (DIL): Increases the degree of membership. E.g., "more or less OLD". (μOLD(x))0.5 (typically power < 1)
  • Concentration (CON): Decreases the degree of membership. E.g., "very OLD". (μOLD(x))2 (typically power > 1)

9. Fuzzy Propositions

A fuzzy proposition is a statement that can have a degree of truth between 0 and 1.

The main difference between classical (crisp) proposition and fuzzy proposition is in the range of their truth values. A classical proposition is either true or false. In fuzzy logic, the truth value of a fuzzy proposition is a matter of degree, expressed by a value T(p) in the interval [0, 1].

Example: "Speed is Slow"

  • T(p) = 0.8, if p is partly true
  • T(p) = 1, if p is absolutely true
  • T(p) = 0, if p is totally false
  1. Unconditional and Unqualified:

    Canonical form: p: V is F

    Where V is a variable, F is a fuzzy set on U. Represents an inaccurate predicate.

    Example: p: Speed (V) is high (F). T(p) = μF(v) where v is the value of V.

  2. Unconditional and Qualified:

    Canonical form: p: (V is F) is S

    Where S is a fuzzy truth qualifier (e.g., "very true", "fairly false").

    Example: Speed is high is very true.

  3. Conditional and Unqualified:

    Canonical form: p: if X is A, then Y is B

    Where X, Y are variables; A, B are fuzzy sets.

    Example: p: if speed is High, then risk is Low.

  4. Conditional and Qualified Propositions:

    Canonical form: p: (if X is A, then Y is B) is S

    Example: p: if speed is high than risk is low is true.

Classical two-valued logic (True/False) can be extended to multi-valued logic. For example, a three-valued logic might denote true (1), false (0), and indeterminacy (1/2).

10. The Linguistic Variable

Linguistic variables are variables whose values are words or sentences in a natural language, rather than numbers.

In mathematics, variables usually take numerical values. In fuzzy logic, non-numeric linguistic variables are often used to define linguistic concepts.

Examples:

  • TEMP can be defined as a linguistic variable with values like {Cold, Warm, Hot}.
  • AGE can have values {Young, Middle-aged, Old}.
  • STATUS can have values {Rich, Middleclass, Poor Hot}.

The values of linguistic variables are called linguistic concepts/terms, and these terms are fuzzy sets.

Formal Definition of Linguistic Variable

Formally, a linguistic variable is characterized by a quintuple (X, T(X), U, G, M) where:

  • X: Name of the variable (e.g., AGE, TEMP).
  • T(X): Set of linguistic terms of X (e.g., {Cold, Warm, Hot} for TEMP).
  • U: Universe of discourse.
  • G: Syntactic rules, which generate terms of T(X).
  • M: Semantic rules associated with each linguistic term of X with its meaning (defines the fuzzy set).

Linguistic Hedges

By modifying linguistic terms using hedges like very, quite, extremely, almost, more or less, etc., we can enhance the utility of linguistic variables. Hedges modify their operands and are valuable in specific situations.

Example: "very Cold", "fairly Warm", "not very Hot and not Cold".

Syntactic rules (G) can generate various valid fuzzy linguistic terms.
Semantic rules (M) give meaning to these terms, e.g., M for "Hot" defines its membership function.

Example Production Rules for Generating Linguistic Terms.

Fuzzy Modifiers

Also called Fuzzy hedges, these modify the value of fuzzy sets. Dilation and concentration are predefined modifiers.

  • Fairly Hot: μfairly_Hot(x) = (μHot(x))0.5 (Dilation)
  • Very Hot: μvery_Hot(x) = (μHot(x))2 (Concentration)
  • Not Very Hot: μnot_very_Hot(x) = 1 - (μHot(x))2

11. Fuzzy Quantifiers

Fuzzy quantifiers are fuzzy numbers that take part in fuzzy propositions. They are of two types:

  1. Type 1: Defined on real numbers and quantified by linguistic terms such as "much more than 5", "at least about 10", "little as 1", etc.
  2. Type 2: Defined on [0, 1] and characterize linguistic terms such as "almost all", "about half", "many", and so on.

Examples:

  • There are about 10 students in the class who are Bright.
  • There are about half people in the state whose income is Low.
  • Almost all people in Europe are having good health.

12. Fuzzy Inference (Reasoning)

Fuzzy inference is the process of deriving conclusions from fuzzy inputs and fuzzy rules.

Inference Rules in Classical Logic (Review)

  • Modus Ponens: Rule: P → Q. Fact: P. Conclusion: Q.
  • Modus Tollens: Rule: P → Q. Fact: ¬Q. Conclusion: ¬P.
  • Chain Rule (Hypothetical Syllogism): Rule1: P → Q. Rule2: Q → R. Conclusion: P → R.

Generalized Inference Rules in Fuzzy Logic

  • Generalized Modus Ponens (GMP):
    Rule: If X is P Then Y is Q
    Fact (Premise): X is P'
    Inference (Conclusion): Y is Q'
    (P, Q, P', Q' are fuzzy sets. Q' is derived from P, Q, P' and the implication relation.)

    Example: Generalized Modus Ponens.

  • Generalized Modus Tollens (GMT):
    Rule: If X is P Then Y is Q
    Fact (Premise): Y is Q'
    Inference (Conclusion): X is P'

    Example: Generalized Modus Tollens.

  • Chain Rule in Fuzzy Logic:
    Rule1: If X is P Then Y is Q
    Rule2: If Y is Q Then Z is R
    Inference: If X is P Then Z is R

Examples:

  1. Rule: If a papaya is yellow then the papaya is ripe. Fact: This papaya is very yellow. Inference: This papaya is very ripe.
  2. Rule: If people are old then they are weak. Fact: Shyam is more or less old. Inference: Shyam is more or less weak.

A fuzzy implication (also known as fuzzy If-Then rule, fuzzy rule, or fuzzy conditional statement) assumes the form:

If x is A then y is B

Where A and B are fuzzy sets defined on the universes of discourses X and Y, respectively.
Often, "x is A" is called the antecedent or premise, while "y is B" is called the consequent or conclusion.

Example: Suppose P and T are universes for pressure and temperature.
Linguistic variables: High temperature (THIGH), Low pressure (PLOW).
THIGH = {(20,0.2), (25,0.4), (30,0.6), (35,0.6), (40,0.7), (45,0.8), (50,0.8)}
PLOW = {(1,0.8), (2,0.8), (3,0.6), (4,0.4)}
Fuzzy implication: If temperature is High then pressure is Low. (R: THIGH → PLOW)

Matrix R for "If temperature is High then pressure is Low".

There are generally two ways to interpret the fuzzy rule "If x is A then y is B":

  1. A coupled with B: This involves T-norm operators. The fuzzy relation R: A → B is defined based on μA(x) and μB(y).

    Common T-norm operators:

    • Minimum (Mamdani rule): Tmin(a, b) = min(a, b) = a ∧ b
      μR(x,y) = min(μA(x), μB(y))
    • Algebraic product (Larsen rule): Tap(a, b) = a ⋅ b
      μR(x,y) = μA(x) ⋅ μB(y)
    • Bounded product: Tbp(a, b) = max(0, a + b - 1)
    • Drastic product: Tdp(a, b) = a if b=1, b if a=1, 0 if a,b < 1

    Interpretation as "A coupled with B" using T-norms.

  2. A entails B: This involves implication functions.
    • Material implication: R: A → B = Ac ∪ B
    • Propositional calculus: R: A → B = Ac ∪ (A ∩ B)
    • Extended propositional calculus: R: A → B = (Ac ∩ Bc) ∪ B (This seems incorrect, typically it's (Ac ∩ Bc) ∪ (A ∩ B) or simply Ac ∪ B)
      More common form is Zadeh's Max-Min Rule for "If X is A THEN Y is B": Rmm = (A × B) ∪ (Ac × Y)
      μRmm(x,y) = max(min(μA(x), μB(y)), 1 - μA(x))
    • Zadeh's arithmetic rule: Rza = Ac ∪ B, interpreted as 1 ∧ (1 - μA(x) + μB(y))
    • Boolean fuzzy rule: Rbf = Ac ∪ B, interpreted as (1 - μA(x)) ∨ μB(x)
    • Goguen's fuzzy rule: Rgf, where μR(x,y) = 1 if μA(x) ≤ μB(y), else μB(y) / μA(x).

    Interpretation as "A entails B" using various implication functions.

    Example for Zadeh's Max-Min Rule: "If x is A THEN y is B" and "If x is A THEN y is B ELSE y is C"

    Zadeh's Max-Min Rule and its application to IF-THEN and IF-THEN-ELSE statements.

13. Fuzzification and Defuzzification

Fuzzification is the process of converting crisp (numerical) input values into fuzzy sets. This involves assigning degrees of membership to these crisp inputs based on predefined membership functions for linguistic terms.

  • It's a method of making crisp values fuzzy, especially when variables are not deterministic and carry uncertainty, imprecision, vagueness, or ambiguity.
  • If data are vague or contain noise, they should be converted into a fuzzy number/set.
  • A fuzzy rule is of the form "If A then B", where A and B are fuzzy propositions. The premise (antecedent) A and consequent B can have multiple parts.
  • The fuzzification module converts the antecedent of a fuzzy rule into a degree of membership between 0 and 1.

Examples:

  • Multiple parts in antecedent: "If sky is dark AND wind is heavy then weather is chilly."
  • "If driving is haphazard then accidents are more AND commuters are not happy."

Example: Fuzzification of Systolic Blood Pressure (BP)

Consider BP = 135 mmHg. Let's use MFs for Low, Normal, and High BP.

Graphs for BP Fuzzy Sets (Low, Normal, High).

From the graph and table: For BP = 135 mmHg:

  • μLow(135) = 0
  • μNormal(135) = 0.35 (approx)
  • μHigh(135) = 0.68 (approx)

Thus, the crisp value BP=135 has been fuzzified into degrees of membership to the fuzzy sets Low, Normal, and High.

Defuzzification is the process of converting a fuzzy output (an aggregated fuzzy set resulting from the inference process) into a single crisp numerical value. This crisp value can then be used for control actions or decisions.

Known defuzzification methods include:

  • Center of Sums (COS) Method:

    The overlapping area in the aggregated output fuzzy set is counted twice.
    X* = (Σ (x̄i ⋅ Areai)) / (Σ Areai) where x̄i is the center of area for the i-th contributing fuzzy set.

    Center of Sums (COS) Method Example.

  • Center of Gravity (COG) / Centroid of Area (COA) Method:

    Calculates the center of the area under the aggregated output fuzzy set.
    For discrete MFs: X* = (Σ (xi ⋅ μA(xi))) / (Σ μA(xi))
    For continuous MFs: X* = (∫ x ⋅ μA(x) dx) / (∫ μA(x) dx)

    Center of Gravity (COG) Method Example.

  • Center of Area / Bisector of Area (BOA) Method:

    Calculates the position under the curve where the areas on both sides are equal.
    ∫αX* μA(x) dx = ∫X*β μA(x) dx (where α and β are the limits of the support).

    COA example calculation table.

  • Weighted Average Method:

    Used for fuzzy sets with symmetrical output MFs; less computationally intensive.
    X* = (Σ (μ(x) ⋅ x)) / (Σ μ(x)) (Here, x is often the peak/center of individual output MFs, and μ(x) is their firing strength/height after aggregation and clipping/scaling).

    Weighted Average Method Example.

  • Maxima Methods: Considers values with maximum membership.
    • First of Maxima (FOM): Smallest value of the domain with maximum membership.
    • Last of Maxima (LOM): Largest value of the domain with maximum membership.
    • Mean of Maxima (MOM): Average of values with maximum membership. X* = (Σ xi∈M xi) / |M| (where M is the set of xi with max membership)

    Example for Maxima Methods (FOM, LOM, MOM).

Lambda-cut (or Alpha-cut) method is applicable to derive a crisp value of a fuzzy set or relation. It converts a fuzzy set/relation into a crisp set/relation for a given value of λ (or α) where 0 ≤ λ ≤ 1.

The λ-cut of a fuzzy set A is Aλ = {x | μA(x) ≥ λ}.

Examples:

A1 = {(x1,0.9), (x2,0.5), (x3,0.2), (x4,0.3)}

  • λ = 0.6: A0.6 = {(x1,1), (x2,0), (x3,0), (x4,0)} which is crisp set {x1}
  • λ = 0.2: A0.2 = {(x1,1), (x2,1), (x3,1), (x4,1)} which is crisp set {x1,x2,x3,x4}

Lambda-cut examples for sets and relations.

Properties of Lambda-cuts:

  • (A ∪ B)λ = Aλ ∪ Bλ
  • (A ∩ B)λ = Aλ ∩ Bλ
  • (Ā)λ ≠ Āλ (complement first then cut, vs cut first then complement of crisp set) - This needs careful interpretation. Typically, (Ac)λ = {x | 1-μA(x) ≥ λ}.

Output of a Fuzzy System (Illustration)

The output of a fuzzy system can be a single fuzzy set or a union of two or more fuzzy sets.
Example: Rules R1: if x is A1 then y is C1; R2: if x is A2 then y is C2.
Output fuzzy set C = C1 ∪ C2.

14. Fuzzy Inference Systems (FIS)

Fuzzy Inference Systems (FIS), also known as fuzzy rule-based systems, fuzzy models, or fuzzy expert systems, are computational frameworks based on fuzzy set theory, fuzzy if-then rules, and fuzzy reasoning.

  • The key unit of a fuzzy logic system is the FIS.
  • The primary work of this system is decision-making.
  • FIS uses "IF...THEN..." rules along with connectors "OR" or "AND" for making necessary decision rules.
  • The input to FIS may be fuzzy or crisp, but the output from FIS is always a fuzzy set.
  • When FIS is used as a controller, it is necessary to have a defuzzification unit for converting fuzzy variables into crisp variables.

Block Diagram of FIS:

Block Diagram of a Fuzzy Inference System.

Components:

  1. Fuzzification Interface Unit: Converts crisp inputs into fuzzy sets.
  2. Knowledge Base:
    • Data Base: Defines the membership functions of the fuzzy sets used in the fuzzy rules.
    • Rule Base: Contains a set of fuzzy IF-THEN rules.
  3. Decision Making Unit (Inference Engine): Performs inference operations on the fuzzy rules to derive fuzzy outputs.
  4. Defuzzification Interface Unit: Converts the fuzzy outputs from the inference engine into crisp values.

There are two main types of Fuzzy Inference Systems:

  1. Mamdani FIS (1975):
    • Uses linguistic fuzzy modeling. The antecedent and consequent of rules are fuzzy sets.
    • Characterized by its high interpretability (rules are easy to understand).
    • Often has lower accuracy compared to Sugeno.
    • Requires defuzzification of the output fuzzy set.
  2. Sugeno FIS (Takagi-Sugeno-Kang or TSK model) (1985):
    • The antecedent of rules is fuzzy, but the consequent is a crisp function (usually linear) of the input variables.
      Rule form: IF x is A AND y is B THEN z = f(x, y) = ax + by + c.
    • Follows precise fuzzy modeling (numerical).
    • Obtains high accuracy but at the cost of low interpretability.
    • The output is already a crisp value (weighted average of individual rule outputs), so defuzzification is simpler or integrated.

In this approach, a rule is composed of fuzzy antecedent and functional consequent parts.

Rule i: IF x1 is A1i AND x2 is A2i ... AND xn is Ani THEN yi = a0i + a1ix1 + ... + anixn

Where aji are coefficients.

The weight of the i-th rule (firing strength) is determined:
wi = μA1i(x1) × μA2i(x2) × ... × μAni(xn) (using product T-norm for AND)

The combined (crisp) output is obtained as a weighted average:

y = (Σ (wi ⋅ yi)) / (Σ wi)

15. Applications of Fuzzy Logic

Introduction

Fuzzy Logic Control (FLC) is a prominent application of fuzzy set theory. It's used in systems that are complex, ill-defined, non-linear, or involve human expertise that is hard to formalize with traditional control methods.

Fig 1: Block Diagram of Closed-loop Control System Design.

Control System Design Steps (General)

  1. Decomposing the large-scale system.
  2. Varying plant dynamics slowly and linearizing.
  3. Organizing state variables, control variables, or output features.
  4. Designing controllers (P, PD, PID, optimal).

In designing a fuzzy logic controller, fuzzy rules play a vital role. The four structures of the fuzzy production rule system (Weiss and Donnel, 1979) are:

  1. A set of rules representing policies and heuristic strategies.
  2. A set of input data assessed prior to the actual decision.
  3. A method for evaluating any proposed action.
  4. A method for generating promising actions and determining when to stop searching.

Architecture and Operations of FLC System

The basic architecture is similar to a general FIS, often including normalization/scaling factors for inputs and outputs.

Fig 2: Basic architecture of a FLC System.

Steps in Designing an FLC:

  1. Locate input, output, and state variables.
  2. Split the complete universe of discourse into fuzzy subsets.
  3. Obtain membership functions for each fuzzy subset.
  4. Assign fuzzy relationships (rule base).
  5. Choose appropriate scaling factors.
  6. Carry out fuzzification.
  7. Identify output contributed from each rule using fuzzy approximate reasoning.
  8. Combine fuzzy outputs obtained from each rule (aggregation).
  9. Finally, apply defuzzification to form a crisp output.

Design Elements for a General FLC System:

  1. Fuzzification strategies and interpretation.
  2. Fuzzy knowledge base: Normalization, partitioning, selection of MFs.
  3. Fuzzy rule base: Selection of inputs/outputs, types of rules, completeness.
  4. Decision-making logic: Fuzzy implication, connective interpretation, inference engine.
  5. Defuzzification materials and interpretation.

FLC systems find a wide range of applications in various industrial and commercial products and systems. They are particularly effective for non-linear, time-varying, ill-defined, and complex systems.

  • Traffic Control
  • Steam Engine Control
  • Aircraft Flight Control
  • Missile Control
  • Adaptive Control
  • Liquid Level Control
  • Helicopter Model Control
  • Automobile Speed Controller / Cruise Control
  • Braking System Controller (ABS)
  • Process Control (e.g., cement kiln, chemical processes)
  • Robotic Control (navigation, manipulation)
  • Elevator (Automatic Lift) Control
  • Automatic Running Control (e.g., trains)
  • Cooling Plant Control
  • Water Treatment Plant Control
  • Boiler Control
  • Nuclear Reactor Control
  • Power Systems Control (e.g., load balancing, fault detection)
  • Air Conditioner Control (Temperature Controller)
  • Control of Biological Processes

  • Fault Detection Control Units
  • Consumer Electronics (washing machines, cameras, rice cookers)
  • Fuzzy Hardware implementation and Fuzzy Computers
