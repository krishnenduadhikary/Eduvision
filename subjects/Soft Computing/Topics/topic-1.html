1. Definition, Scope, and Goal

Definition of Soft Computing (SC)

Lotfi A. Zadeh, 1992: "Soft Computing is an emerging approach to computing which parallel the remarkable ability of the human mind to reason and learn in an environment of uncertainty and imprecision."

Scope

Soft computing refers to principle components like fuzzy logic, neural networks, and genetic algorithms, which have their roots in Artificial Intelligence. The healthy integration of all these techniques has resulted in extending the capabilities of the technologies to more effective and efficient problem solving methodologies.

Goal

  • The main goal of Soft Computing is to develop intelligent machines to provide solutions to real world problems, which are not modeled, or too difficult to model mathematically.
  • Its aim is to exploit the tolerance for Approximation, Uncertainty, Imprecision, and Partial Truth in order to achieve close resemblance with human like decision making.

2. Components of Soft Computing

The main components of soft computing are fuzzy logic, neural networks, evolutionary computation, and probabilistic reasoning. These techniques can be used individually or combined to create powerful systems that mimic human reasoning. Here's a more detailed explanation of each component:

1. Fuzzy Logic (FL)

Purpose: Deals with uncertainty and imprecision by allowing for partial truths and approximate reasoning.

How it works: Introduces membership degrees to represent the truth value of a statement, rather than just true or false.

Examples: Used in control systems (e.g., washing machines, anti-lock braking systems), decision-making, and image processing.

2. Neural Networks (NN)

Purpose: Inspired by the structure and function of the human brain, used for pattern recognition and learning.

How it works: Mimics the way neurons in the brain process information, with interconnected nodes (neurons) that learn from data through processes like backpropagation.

Examples: Used in image recognition (e.g., facial recognition), natural language processing (e.g., translation), and financial modeling (e.g., stock market prediction).

3. Genetic Algorithm (GA)

Purpose: Solves optimization and search problems by mimicking the process of natural selection.

How it works: Uses techniques such as selection, crossover, and mutation to evolve a population of candidate solutions toward an optimal solution.

Examples: Applied in scheduling (e.g., job-shop scheduling), engineering design (e.g., structural optimization), and machine learning (e.g., feature selection).

4. Evolutionary Computation (EC)

Purpose: Uses evolutionary principles (like natural selection, mutation, and crossover) to solve complex optimization problems.

How it works: Employs algorithms like genetic algorithms, evolutionary programming, and genetic programming that simulate the evolution of a population of solutions to find the best one(s).

Examples: Used in design optimization (e.g., aerospace engineering), scheduling (e.g., job-shop scheduling), and machine learning (e.g., optimizing neural network architectures).

5. Probabilistic Reasoning (PR)

Purpose: Deals with uncertainty by representing knowledge and relationships using probabilities.

How it works: Uses techniques like Bayesian networks and Hidden Markov Models to model uncertain relationships between variables and make inferences.

Examples: Used in medical diagnosis (e.g., expert systems), risk assessment (e.g., financial risk), and data analysis (e.g., spam filtering).

3. Hybrid Computing

Hybrid computing represents a powerful synergy by combining conventional hard computing techniques with emerging soft computing methodologies.

  • A part of a complex real-world problem will be solved using hard computing and the remaining part can be tackled utilizing soft computing.
  • In this context, hard computing and soft computing are complementary to each other, each bringing its strengths to solve multifaceted problems more effectively.

4. Hard Computing vs. Soft Computing

Understanding the distinctions between hard and soft computing is crucial for appreciating the unique contributions of soft computing approaches.

Hard Computing Soft Computing
It requires a precisely stated analytical model and often a lot of computation time. It is tolerant of imprecision, uncertainty, partial truth, and approximation.
It is based on binary logic, crisp systems, numerical analysis and crisp software. It is based on fuzzy logic, neural nets and probabilistic reasoning.
It has the characteristics of precision and categoricity. It has the characteristics of approximation and dispositionality.
Hard Computing Soft Computing
It is deterministic. It incorporates stochasticity.
It requires exact input data. It can deal with ambiguous and noisy data.
It is strictly sequential. It allows parallel computations.
It produces precise answers. It can yield approximate answers.

  • Solving problems that cannot be easily modelled mathematically.
  • Addressing non-linear problems effectively.
  • Incorporating human-like knowledge (cognition, learning, recognition) into computing.
