Introduction to Sequential Circuits

Sequential circuits are a type of digital logic circuit whose output depends not only on the present value of its input signals but also on the sequence of past inputs, i.e., its previous state. This is in contrast to combinational circuits, whose outputs are a direct function of only the present inputs.

Key Characteristics:

  • Memory Elements: Sequential circuits contain memory elements (like flip-flops or latches) to store information about past states.
  • Feedback: They often employ feedback paths, where some outputs are fed back as inputs.
  • Clock Signals: Most sequential circuits are synchronous, meaning their state changes are synchronized by a periodic clock signal. Asynchronous sequential circuits also exist but are less common for complex designs.
  • Used for building counters, registers, memory units, and finite state machines.

Flip-Flops: The Basic Memory Element

A flip-flop is a bistable multivibrator, meaning it has two stable states (0 and 1) and can be used to store one bit of binary information. It is the fundamental building block of many sequential circuits.

Common Types of Flip-Flops.

Flip-flops change state based on their inputs and, for clocked types, a clock signal. Common types include SR, JK, D, and T flip-flops.

SR Flip-Flop (Set-Reset)

The SR flip-flop, also known as an SR Latch (if unclocked), is a basic one-bit memory device. It has two inputs: S (Set) and R (Reset).

  • S input sets the output Q to 1.
  • R input resets the output Q to 0.
  • If both S and R are 0, the flip-flop maintains its current state.
  • The condition S=1 and R=1 is typically considered invalid or undefined as it leads to an unpredictable state or oscillation.

JK Flip-Flop

The JK flip-flop is an improvement over the SR flip-flop as it does not have an invalid input state. It has J (Set-like) and K (Reset-like) inputs, and a clock input.

  • If J=0, K=0: No change in state (Qnext = Q).
  • If J=0, K=1: Reset state (Qnext = 0).
  • If J=1, K=0: Set state (Qnext = 1).
  • If J=1, K=1: Toggle state (Qnext = Q'). This is the key advantage.

Master-Slave JK Flip-Flop: A common construction method to avoid race-around conditions (where the output toggles multiple times during a single clock pulse if J=K=1). It consists of two cascaded latches: a master latch that receives inputs during one phase of the clock, and a slave latch that updates its output from the master during the other clock phase.

D Flip-Flop (Data or Delay)

The D flip-flop has a single data input D and a clock input. Its output Q takes the value of the D input at the active edge of the clock pulse. It is commonly used for storing data (registers) and delaying signals.

Characteristic Equation: Qnext = D

T Flip-Flop (Toggle)

The T flip-flop has a single input T (Toggle) and a clock input. If T=0, the flip-flop holds its state. If T=1, the flip-flop toggles its state on the active clock edge.

It can be derived from a JK flip-flop by connecting J and K inputs together to form T.

Characteristic Equation: Qnext = T ⊕ Q = T'Q + TQ'

Edge-Triggered vs. Level-Triggered:

  • Level-Triggered: The flip-flop (often a latch) responds to inputs as long as the clock signal is at a certain level (e.g., HIGH).
  • Edge-Triggered: The flip-flop responds to inputs only at the transition of the clock signal (either positive/rising edge or negative/falling edge). Most modern flip-flops are edge-triggered to ensure predictable state changes.

Flip-Flop Conversions

It's often necessary to convert one type of flip-flop into another using additional combinational logic. The general steps involve understanding the source and target flip-flops, using their characteristic tables/equations, and deriving the logic for the source flip-flop's inputs to mimic the target's behavior.

D to T Flip-Flop

A D flip-flop can function as a T flip-flop by connecting the T input to an XOR gate whose other input is Q, and the output of the XOR gate to D. So, D = T ⊕ Q. No additional combinational logic is required if T input is available and Q output can be fed back.

Counters

A counter is a sequential circuit that goes through a predetermined sequence of states upon the application of input pulses (usually clock pulses). Counters are essential for timing, sequencing, and frequency division.

Counter Classification:

  • Asynchronous (Ripple) Counters: The flip-flops are not triggered by a common clock signal. The output of one flip-flop triggers the clock input of the next.
  • Synchronous Counters: All flip-flops are triggered simultaneously by a common clock signal.

Counters can also be characterized by:

  • Up, Down, or Up/Down counting capability.
  • Positive or Negative edge triggering.
  • The number of states (modulus of the counter).

Asynchronous (Ripple) Counters

In asynchronous counters, only the first flip-flop is driven by the main clock. The clock input of subsequent flip-flops is driven by the output of the previous flip-flop. This creates a "ripple" effect as the count propagates through the stages.

Synchronous Counters

In synchronous counters, all flip-flops share a common clock input, and their state changes are synchronized. This eliminates the cumulative propagation delay found in ripple counters, allowing for higher frequency operation.

Special Counters

Ring Counter

A ring counter is a circular shift register where only one flip-flop is set at any particular time, and all others are cleared. The '1' bit is shifted from one flip-flop to the next on each clock pulse, creating a sequence of timing signals.

Ring Counter operation and generated timing signals.

Johnson Counter (Twisted Ring Counter)

A Johnson counter is a modified ring counter where the complemented output of the last flip-flop is fed back to the input of the first. This creates a sequence of 2k states for k flip-flops.

Shift Registers

A shift register is a type of sequential logic circuit, mainly for storage and transfer of digital data. They are a group of flip-flops connected in a chain so that the output from one flip-flop becomes the input of the next flip-flop. Most of the registers possess no characteristic internal sequence of states.

General concept of a Shift Register.

  • A simple shift register can be made using D-type flip-flops.
  • Data is moved ("shifted") to their required positions on each clock pulse.
  • Shift registers can operate in various modes based on how data is loaded and retrieved.

Types of Shift Registers:

  • SISO (Serial-In, Serial-Out): Data is shifted in one bit at a time and shifted out one bit at a time.
  • SIPO (Serial-In, Parallel-Out): Data is shifted in serially, but all bits are available simultaneously as parallel output.
  • PISO (Parallel-In, Serial-Out): Data is loaded in parallel simultaneously into all flip-flops, and then shifted out serially.
  • PIPO (Parallel-In, Parallel-Out): Data is loaded in parallel and retrieved in parallel. Often used for temporary storage.

Bidirectional and Universal Shift Registers:

  • Bidirectional Shift Register: Can shift data either left or right.
  • Universal Shift Register: Combines capabilities for serial/parallel input/output and left/right shifting.

Applications of Shift Registers:

  • Temporary data storage.
  • Serial-to-parallel and parallel-to-serial data conversion.
  • Time delay generation.
  • Sequence generation (e.g., Ring and Johnson counters).
  • Arithmetic operations (multiplication/division by powers of 2).
