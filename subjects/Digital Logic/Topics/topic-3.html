Introduction to Logic Gates

Logic gates are the most basic building blocks of any digital system. They are electronic circuits (hardware) that have one or more inputs and typically one output. The relationship between the input and the output is based on a certain logic.

  • A logic gate is a piece of hardware or an electronic circuit used to implement basic logic expressions.
  • They perform logical operations on one or more binary inputs to produce a single binary output (0 or 1, LOW or HIGH).
  • The three fundamental logic gates are: OR gate, AND gate, and NOT gate.

Basic Logic Gates

AND Gate

A logic AND gate is a basic logic gate whose output is equal to the logical product (multiplication) of its inputs. The output of an AND gate is HIGH (1) only if all of its inputs are HIGH (1). For all other cases, the output is LOW (0).

OR Gate

The output of the logic OR gate is equal to the logical sum (addition) of its inputs. The output of an OR gate is HIGH (1) if any of its inputs are HIGH (1). The output is LOW (0) only when all the inputs are LOW (0).

NOT Gate (Inverter)

A logic NOT gate, also known as an inverter, is a basic logic gate whose output is equal to the inverse (complement) of its single input. If the input is HIGH (1), the output is LOW (0), and if the input is LOW (0), the output is HIGH (1).

Universal Logic Gates

Universal logic gates are gates that can be used to implement any other logic gate or any Boolean function without needing any other type of gate. The two universal logic gates are NAND and NOR.

  • The reason they are universal is that NAND gates and NOR gates can perform (or can function like) all three basic gates: AND, OR, and NOT.
  • We can design any basic logic gate or complex Boolean expression by using only NAND gates or only NOR gates.

NAND Gate

A NAND (NOT-AND) gate is a logical combination of an AND gate followed by a NOT gate. Its output is the complement of the AND operation. The output is LOW (0) only if all inputs are HIGH (1); otherwise, the output is HIGH (1).

Boolean Expression: Y = (A ¬∑ B)' or Y = (AB)'

NOR Gate

A NOR (NOT-OR) gate is a logical combination of an OR gate followed by a NOT gate. Its output is the complement of the OR operation. The output is HIGH (1) only if all inputs are LOW (0); otherwise, the output is LOW (0).

Other Common Logic Gates

XOR (Exclusive-OR) Gate

The Exclusive-OR (XOR) gate produces a HIGH (1) output only if its inputs are different (one is HIGH, the other is LOW). If the inputs are the same (both HIGH or both LOW), the output is LOW (0).

XNOR (Exclusive-NOR) Gate

The Exclusive-NOR (XNOR) gate is the complement of the XOR gate. It produces a HIGH (1) output only if its inputs are the same (both HIGH or both LOW). If the inputs are different, the output is LOW (0). It is also known as an equivalence gate.

Gate-Level Implementation of Boolean Expressions

Any Boolean expression can be implemented by a network of logic gates. The expression dictates the types of gates needed and how they are interconnected.

  • Product terms (AND operations) are implemented using AND gates.
  • Sum terms (OR operations) are implemented using OR gates.
  • Complements (NOT operations) are implemented using NOT gates (inverters).

Example: Implement F = AB + C'D

  1. One AND gate for AB.
  2. One NOT gate for C'.
  3. One AND gate for C'D (inputs C' and D).
  4. One OR gate for (AB) + (C'D).

Circuit for F = AB + C'D
Inputs A, B -> AND gate 1 (Output AB)
Input C -> NOT gate (Output C')
Inputs C', D -> AND gate 2 (Output C'D)
Outputs of AND gate 1 and AND gate 2 -> OR gate (Output F)

Conceptual logic circuit for F = AB + C'D.

Using universal gates (NAND-only or NOR-only implementations) is also very common, often leading to more efficient circuit designs in certain technologies.
