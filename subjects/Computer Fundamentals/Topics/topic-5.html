Memory and Storage

Understanding how computers store data and instructions, from fast, temporary primary memory to capacious, permanent secondary storage.

Introduction to Computer Memory

The computer's memory stores data, instructions required during the processing of data, and output results. Storage may be required for a limited period of time, instantly, or for an extended period of time. Different types of memories, each having its own unique features, are available for use in a computer. The cache memory, registers, and RAM are fast memories and store the data and instructions temporarily during the processing of data and instructions. The secondary memory like magnetic disks and optical disks have large storage capacities and store the data and instructions permanently, but are slow memory devices. The memories are organized in the computer in a manner to achieve high levels of performance at the minimum cost.

3.3 Memory Hierarchy

The memory is characterized on the basis of two key factorsâ€”capacity and access time. Capacity is the amount of information (in bits) that a memory can store. Access time is the time interval between the read/write request and the availability of data. The lesser the access time, the faster is the speed of memory. Ideally, we want the memory with fastest speed and largest capacity. However, the cost of fast memory is very high. The computer uses a hierarchy of memory that is organized in a manner to enable the fastest speed and largest capacity of memory. The hierarchy of the different memory types is shown in Figure 3.2.

Memory Hierarchy Diagram

The internal memory and external memory are the two broad categories of memory used in the computer. The internal memory consists of the CPU registers, cache memory and primary memory. The internal memory is used by the CPU to perform the computing tasks. The external memory is also called the secondary memory. The secondary memory is used to store the large amount of data and the software.

Primary Memory

Primary memory is the main memory of computer. It is a chip mounted on the motherboard of computer. Primary memory is categorized into two main types: Random Access Memory (RAM) and Read Only Memory (ROM).

RAM is used for the temporary storage of input data, output data and intermediate results. The input data entered into the computer using the input device, is stored in RAM for processing. After processing, the output data is stored in RAM before being sent to the output device. Any intermediate results generated during the processing of program are also stored in RAM. Unlike RAM, the data once stored in ROM either cannot be changed or can only be changed using some special operations. Therefore, ROM is used to store the data that does not require a change. Flash memory is another form of rewritable read-only memory that is compact, portable, and requires little energy.

3.3 Random Access Memory (RAM)

Random Access Memory (RAM) is a volatile memory and loses all its data when the power is switched off. It is the main memory of the computer system that stores the data temporarily and allows the data to be accessed in any order. As compared to the secondary storage, the data can be accessed at a faster speed in RAM because it is the internal memory of the computer. Figure 3.3 shows RAM with ICs.

RAM Stick Image

RAM is made up of different ICs, which are mounted on a printed circuit board. RAM stores the application programs and the data on which the user is currently working so that the processor can easily access the required application program and data in a less amount of time. RAM is also known as read/write memory because it can perform both read as well as write operations. The speed of RAM is faster than the other memory devices, such as hard disk, floppy disk, etc.

The programs, which are being currently executed by the computer system, are stored in RAM. RAM is volatile and, therefore, the programs and the data stored in the RAM get lost when the power supply is switched off. The storage capacity of RAM is usually less than the secondary storage device. RAM can be categorised into two main types, namely, Static RAM (SRAM) and Dynamic RAM (DRAM), which can be further divided into various types for storing data. 3.3.1 Static RAM (SRAM)

Static RAM (SRAM) is a type of RAM in which data is stored till the power of the computer system is switched on. SRAM uses a number of transistors to store a single bit of digital information. Figure 3.5 shows the organisation of data in a cell of SRAM.

SRAM Cell Diagram

SRAM can be divided into the following three types: Asynchronous SRAM (ASRAM), Burst SRAM (BSRAM), and Pipeline Burst SRAM (PBSRAM).

  • Asynchronous SRAM (ASRAM): Performs its operations without the use of system clock. Uses signals like Chip Select (CS), Write Enable (WE), and Output Enable (OE).
  • Burst SRAM (BSRAM): Works in association with the system clock and is also known as synchronous SRAM. Commonly used with high-speed applications.
  • Pipeline Burst SRAM (PBSRAM): Makes use of the pipeline technology in which a large amount of data is broken up in the form of different packets containing data. Can operate at bus rates as high as 66 MHz.

3.3.2 Dynamic RAM (DRAM)

DRAM is the RAM in which data is stored in a storage cell, consisting of a transistor and a capacitor. Unlike SRAM, the DRAM needs to be continuously refreshed with power supply because the capacitor has the tendency to get discharged. DRAM retains the data for a very short span of time, even after the power supply is switched off. Figure 3.6 shows the organisation of data in a cell of DRAM.

DRAM Cell Diagram

DRAM can be divided into the following types:

  • Synchronous DRAM (SDRAM): Performs its operations in the synchronous mode, i.e., in association with the clock cycle of the processor bus.
  • Rambus DRAM (RDRAM): Designed by Rambus Inc. works at a faster speed, as compared to SDRAM. Compact in size and uses 16-bit address bus.
  • Extended Data Out DRAM (EDODRAM): Can access more than one bit of data at one time which helps in achieving faster data access rates.
  • Fast Page Mode DRAM (FPMDRAM): Makes use of paging in which read or write operation is performed by selecting the address of the data from the rows and the columns of a matrix.

3.4 Read Only Memory (ROM)

ROM is the memory that stores the data permanently, i.e., it can retain the data even when the power of the computer system is switched off. The data can be easily read from this type of memory but cannot be changed. ROM is most commonly used in devices such as calculators, laser printers, etc. ROM does not allow the random access of data rather it allows sequential access of data. It is less expensive as compared to RAM and other storage devices, such as magnetic disk, etc. ROM is divided into four types, which are as follows: Programmable ROM (PROM), Erasable PROM (EPROM), Electrically Erasable PROM (EEPROM), Flash ROM.

  • 3.4.1 Programmable ROM (PROM): A memory chip on which the write operation of data can be performed only once. Once programmed, it cannot be erased or destroyed.
  • 3.4.2 Erasable PROM (EPROM): Can be erased or destroyed using Ultraviolet Light (UL). Can be reprogrammed.
  • 3.4.3 Electrically Erasable PROM (EEPROM): Can be erased or destroyed by exposing it to an electric charge. Stores data permanently but allows changes. Can be erased one byte at a time.
  • 3.4.4 Flash ROM (Flash Memory): A type of EEPROM that stores information using floating-gate transistors. Faster speed of reading data. Used in mobile phones, digital cameras, iPods. Memory cells are made of floating-gate transistors.
    • NAND flash: Each cell behaves like a MOSFET.
    • NOR flash: Provides facility to write data using tunnel injection and erase using tunnel release.

Secondary Memory (Storage)

Secondary memory, also known as secondary storage, refers to the storage devices and systems used to store data persistently, even when the computer is powered off. Unlike primary memory (RAM), which is fast and temporary, secondary memory is slower but offers much larger storage capacities. These devices are essential for long-term data storage and retrieval.

Secondary Memory Examples Image

Use of Secondary Memory

  • Permanent storage: Stores data permanently, even if power is off.
  • Large Storage: Provides large space for videos, images, audios, files, etc.
  • Portable: Some devices are removable for easy data transfer.

Types of Secondary Memory

There are two types of secondary memory: Fixed Devices and Removable Devices.

Secondary Memory Types Diagram

1. Fixed Devices

Permanently installed storage devices, like internal hard drives or SSDs. Data is always accessible.

  • Hard Disk Drive (HDD): Traditional storage, data on spinning magnetic disks. Slower, but offers large space at low cost.
  • Solid-State Drive (SSD): Faster, newer type using flash memory. No moving parts, more reliable and faster.
  • External Hard Drives (if kept permanently connected): Technically external, but can be considered fixed if always connected.
  • Network Attached Storage (NAS): Storage device connected to a network, allowing multiple users access. Used for backups and file sharing.

2. Removable Devices

Can be easily disconnected and used on different systems. Allow for easy data transfer and backup.

  • Optical Discs (CD, DVD, Blu-ray):
    • CD (Compact Disc): Holds up to 700 MB.
    • DVD (Digital Versatile Disc): Typically 4.7 GB or more.
    • Blu-ray Disc: 25 GB (single layer) to 50 GB (dual layer).
  • USB Flash Drives (Pen Drives): Small, portable, use flash memory. Capacities from gigabytes to terabytes.
  • Magnetic Tapes: Older form, thin tapes. Still used for large-scale data storage and archiving.
  • Flash Memory Cards (SD Cards, MicroSD Cards): Tiny, portable, used in cameras, smartphones, etc.
  • External Hard Drives (if used as removable): Commonly used for backup or transferring large files.
  • Cloud Storage: Not a physical device but a service (Google Drive, Dropbox) storing data online on servers.

Applications of Secondary Memory

  • Data Storage & Archiving
  • Backup & Recovery
  • Software & OS Storage
  • Media & Content Storage
  • Database Management
  • Virtual Memory
  • Cloud Storage (as an extension)
  • File Sharing
  • Gaming
  • Business & Research

Advantages of Secondary Memory

  • Large storage capacity
  • Non-volatile storage
  • Portability (for many types)
  • Cost-effective (generally)

Disadvantages of Secondary Memory

  • Slower access times (than primary memory)
  • Mechanical failures (e.g., HDDs)
  • Limited lifespan (write cycles)
  • Data corruption (EMI, viruses, physical damage)

Size The computer has a small primary memory. The computer has a larger secondary memory.
Expense Primary memory is costlier than Secondary. Secondary Memory is cheaper than primary.
