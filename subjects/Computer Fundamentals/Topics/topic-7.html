Software and Hardware

Delving into the essential components that make a computer function: the physical hardware and the instructive software, including their types and interplay.

Introduction

A computer is an electronic device that accepts data (input) and, processes data arithmetically and logically, to produce information (output). It is divided into two main categories – Hardware & Software.

Relationship between Hardware and Software

  • Hardware and software are mutually dependent on each other. Both of them must work together to make a computer produce a useful output.
  • Software cannot be utilized without supporting hardware.
  • Hardware without a set of programs to operate upon cannot be utilized and is useless.
  • To get a particular job done on the computer, relevant software should be loaded into the hardware.
  • Hardware is a one-time expense.
  • Software development is very expensive and is a continuing expense.
  • Different software applications can be loaded on a hardware to run different jobs.
  • A software acts as an interface between the user and the hardware.
  • If the hardware is the ‘heart’ of a computer system, then the software is its ‘soul’. Both are complementary to each other.

Hardware

Hardware is the physical aspect of computers, telecommunications, and other devices. It includes not only the computer proper but also the cables, connectors, power supply units, and peripheral devices such as the keyboard, mouse, audio speakers, and printers.

Types of Hardware

Internal Hardware (Components)

Internal hardware includes:

  • a) Central Processing Unit (CPU): It is the brain of the computer. Most of calculations take place here. Two typical components of a CPU are - Arithmetic Logic Unit (ALU) and Control Unit (CU).
  • b) Motherboard: Is the main circuit board of a microcomputer. It contains the CPU, memory, expansion slots and all the controllers required to control standard hardware devices.
  • c) Random Access Memory (RAM): RAM is the place in a computer where the operating system, application programs, and data in current use are kept. It is a temporary memory and can be compared to a person’s short-term memory.
  • d) Read Only Memory (ROM): It is a technology that allows you to write data only once. After the data has been written, you can read it an unlimited number of times.

Input Hardware

Input hardware consists of external devices by which we give information and instructions to the computer. Common input hardware are:

  • a) Keyboard: The set of typewriter-like keys that enables you to enter data into a computer.
  • b) Mouse: A device that controls the movement of the cursor or pointer on a display screen.
  • c) Scanner: A device that can read text or illustrations printed on paper and translate the information into a form the computer can use.

Output Hardware

Output hardware consists of internal and external devices that transfer information from the computer’s CPU to the computer user. Common output hardware are:

  • a) Monitor: A display screen used to present output from a computer, video camera, VCR or other video generator.
  • b) Printer: In computers, a printer is a device that accepts text and graphic output from a computer and transfers the information to paper.
  • c) Speaker: Speakers or headphones to an output port to hear the audio produced by the computer.

Connecting Hardware

Connecting devices allow hardware devices to communicate with each other. Common devices are:

  • a) Modem: Modems connect a personal or portable computer to dial-up networks through a regular telephone line.
  • b) LAN Card: A Local Area Network (LAN) card is used to provide wireless Internet access to computer users in home or roaming networks.

Storage Hardware

A computer storage device is any type of hardware that stores data. Some common storage devices are:

  • a) Hard Disc: A magnetic disk on which you can store computer data. The term hard is used to distinguish it from a soft, or Floppy disk.
  • b) Floppy Disc: A flexible plastic disk coated with magnetic material and covered by a protective jacket, used primarily by computers to store data magnetically.
  • c) CD ROM: A CD-ROM is a CD that can be read by a computer with an optical drive.

Software

A set of instructions that enables physical components of a computer to work in a synchronized way is known as Software. This set of instructions is often called a program. Computers cannot do any useful work without instructions from software.

Types of Software

Computer software is also divided in four major types:

  • a) System Software
  • b) Programming Software
  • c) Application Software
  • d) Utility Software

a) System Software

System software helps in running the computer hardware and the computer system. Main system softwares are:

  • i) Device Drivers: In computing, a device driver is a computer program allowing higher-level computer programs to interact with a hardware device.
  • ii) Operating System: An operating system is an interface between hardware and user; it is responsible for the management and coordination of activities and the sharing of the limited resources of the computer.

b) Application Software

It enables the users to accomplish certain specific tasks. Following are the types of application software:

  • i) Web Browsing Software: It allows one to surf the Web. Example: MS Internet Explorer.
  • ii) Word Processing Software: Allows users to create, edit a document. Example: MS Word.
  • iii) Spreadsheet Software: Allows users to create document and perform calculation. Example: MS Excel etc.
  • iv) Database Software: Allows users to store and retrieve vast amount of data. Example: MS Access etc. (Note: Roman numeral iv was used in image, consistent here)
  • v) Presentation Graphics Software: Allows users to create visual presentation. Example: MS PowerPoint. (Note: Changed from ii to v based on apparent sequence in image)

c) Utility Software

Utility software helps in the management of computer hardware and application software. It performs a small range of tasks. Some typical examples of utility software are:

  • i) Disk Defragmentors: It detects computer files whose contents are stored on the hard disk in disjointed fragments, and moves the fragments together to increase efficiency.
  • ii) Backup: This utility makes a copy of all information stored on a disk, and restores either the entire disk or selected files.
  • iii) Disk Partitioner: It divides an individual drive into multiple logical drives, each with its own file system which can be mounted by the operating system and treated as an individual drive.
  • iv) Anti-Virus: These utilities scan computer for viruses.

d) Programming Software

This is one of the most commonly known and popularly used forms of computer software. These software come in forms of tools that assist a programmer in writing computer Programs. The tools that help the programmers in instructing a computer system include:

  • a) Text Editor
  • b) Compilers
  • c) Interpreters

Open Source vs Proprietary Software

Open Source Software (OSS)

Open source software (OSS) is distributed under a licensing agreement which allows computer code to be shared, viewed and modified by other users and organizations. Open Source is when the source code is available for anyone to use. They can change and share it. Everyone has access to open-source software and it is free to download. This means anyone can use the software, modify it and then share it among the community.

Examples of Open Source:

  • WordPress - A Content Management System (CMS) used for blogs and creating our websites.
  • GitHub - A hosting service that has many of its features.
  • Mozilla Firefox - A web browser and one of the most popular browsers available.
  • Linux - One of the most commonly used operating systems. Linux is found on desktops, mobile phones and a large number of servers.
  • Android - An operating system that controls the majority of the world’s smartphones, it is also based on Linux.
  • GIMP - An example of a free alternative to proprietary software, GIMP is a popular alternative to Adobe Photoshop.

Proprietary Software ("Closed Source")

Proprietary Software or “Closed Source” is different to open source. Instead of being free for anyone to use, proprietary software is owned by an individual or company. The source code is usually kept secret. This means only people with access to the code, can modify and change it. Proprietary software is also, but not always, paid software.

Examples of Proprietary Software:

  • Microsoft Office – Productivity software including a text editor, powerpoint creator and more.
  • Adobe Photoshop – An image creation and manipulation program used for digital artwork.
  • Windows – The most widely used operating system in the world.
  • Internet Security Programs – Designed to keep our computer safe from online threats.
  • Shopify – An eCommerce website where we can build your online shop.
  • Big Commerce – Like Shopify, it is an online store builder.
  • EKM – Platform for creating an online shop.

Differences between Open Source and Closed Source Software

  • Price Policy: OSS often free, may have costs for extras. Closed source usually paid, higher price but often more features/support.
  • Security: OSS code viewable/modifiable by community (quick bug fixes). Closed source fixed by vendor (can take longer).
  • Quality of Support: Closed source usually offers direct support. OSS support via forums, articles, hired help (can be less direct).
  • Source Code Availability: OSS allows source code changes. Closed source restricted.
  • Usability: OSS guides can be developer-focused. Closed source often has better user-focused documentation.

Firmware Basics

Firmware is a type of software embedded within hardware devices, acting as a bridge between the hardware and the operating system. It provides low-level control over hardware, enabling devices to function, communicate, and perform basic tasks. Think of it as the instructions that tell a printer how to print, or a hard drive how to store data.

  1. What is Firmware?

    Firmware is software that is stored in a device’s non-volatile memory (like ROM, EPROM, or flash memory). It’s designed to control the hardware of a specific device, rather than being general-purpose software like applications. It’s crucial for device functionality, including starting up, communicating with other devices, and performing basic input/output operations.

  2. Examples of Firmware:
    • BIOS/UEFI: Found on computer motherboards, manages initial startup and hardware configuration.
    • Printer Firmware: Controls how a printer interacts with the computer and performs printing tasks.
    • Hard Drive/SSD Firmware: Manages the storage and retrieval of data on hard drives and SSDs.
    • Network Device Firmware: Controls the functionality of routers, switches, and other network hardware.
    • Peripheral Device Firmware: Found in devices like USB flash drives, keyboards, mice, and sound cards.
  3. Key Differences Between Firmware and Software:
    • Location: Firmware is embedded within the hardware, while software is generally stored separately on a storage device like a hard drive.
    • Functionality: Firmware focuses on low-level hardware control, while software handles more general-purpose tasks and applications.
    • Modifiability: Firmware is usually less easily updated than software, though it can be updated through firmware upgrades.
  4. Why is Firmware Important?
    • Hardware Initialization: Firmware initializes the hardware when a device starts up.
    • Device Communication: It enables devices to communicate with each other and with the operating system.
    • Basic Input/Output: Firmware handles essential input/output tasks, like reading data from a keyboard or sending data to a monitor.
    • Device Control: It provides low-level control over the device’s hardware components.
