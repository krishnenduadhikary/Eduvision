Arrays

Arrays are fundamental data structures in computer programming that store a collection of elements of the same data type under a single name. These elements are stored in contiguous memory locations, and each element is accessed using an index or a subscript.

Key Characteristics of Arrays Understanding these core concepts will help you master array operations:

Contiguous Memory Fixed Size Random Access Homogeneous Elements

Array Basics

Declaration & Initialization

// C++ array declaration

int numbers[5]; // Declaration

int values[] = {10, 20, 30}; // Initialization


// Java array declaration

int[] numbers = new int[5];

int[] values = {10, 20, 30};

Arrays can be declared with or without initialization. The size must be specified (either explicitly or implicitly during initialization).

Array Operations

Traversal

Systematically visiting each element within the array using loops.

Forward Traversal

for(int i = 0; i < length; i++) {

// Access array[i]

}

Backward Traversal

for(int i = length-1; i >= 0; i--) {

// Access array[i]

}

Insertion

Adding an element at a specific position requires shifting subsequent elements.

Insertion Steps
  1. Check if array has space
  2. Shift elements from insertion point to right
  3. Insert new element
  4. Update array length
Example Code

// Insert 6 where element > 5

for(int i = 0; i < length; i++) {

if(array[i] > 5) {

// Shift elements

for(int j = length; j > i; j--) {

array[j] = array[j-1];

}

array[i] = 6; // Insert

length++;

break;

}

}

Deletion

Removing an element requires shifting subsequent elements to fill the gap.

Deletion Steps
  1. Find element to delete
  2. Shift subsequent elements left
  3. Decrement array length
  4. Adjust index if needed
Example Code

// Delete elements > 5

for(int i = 0; i < length; i++) {

if(array[i] > 5) {

// Shift elements

for(int j = i; j < length-1; j++) {

array[j] = array[j+1];

}

length--;

i--; // Recheck current position

}

}

Searching

Finding an element in an array can be done through:

  • Linear Search: Check each element sequentially
  • Binary Search: For sorted arrays (much faster)

// Linear search example

for(int i = 0; i < length; i++) {

if(array[i] == target) {

return i; // Found at index i

}

}

Updating

Modifying an element at a specific position is straightforward with direct access.

// Update element at index 2

array[2] = newValue;


// Conditional update example

for(int i = 0; i < length; i++) {

if(array[i] == oldValue) {

array[i] = newValue;

}

}

Address Calculation

The address of an element in a 2D array can be calculated using these formulas:

Row-Major Formula

Address = B + W × [N × (I - Lr) + (J - Lc)]

  • B = Base address
  • W = Element size (bytes)
  • N = Number of columns
  • I = Row index
  • J = Column index
  • Lr = Lower row bound (usually 0)
  • Lc = Lower column bound (usually 0)
Column-Major Formula

Address = B + W × [(I - Lr) + M × (J - Lc)]

  • B = Base address
  • W = Element size (bytes)
  • M = Number of rows
  • I = Row index
  • J = Column index
  • Lr = Lower row bound (usually 0)
  • Lc = Lower column bound (usually 0)
Example Calculation

Given array B[1300..1900] with base address 1020 and element size 2 bytes, find B[1700]:

Address = 1020 + 2 × (1700 - 1300) = 1020 + 800 = 1820

This calculation works for both row-major and column-major in 1D arrays.

Applications & Limitations

Applications

  • Mathematical computations: Matrices, vectors
  • Image processing: Pixel data storage
  • Game development: Game boards, maps
  • Database systems: Table representations
  • CPU scheduling: Process queues
  • Sorting algorithms: QuickSort, MergeSort
  • Lookup tables: Efficient data retrieval

Limitations

  • Fixed size: Cannot grow/shrink dynamically
  • Memory waste: May allocate unused space
  • Insertion/deletion: Costly operations due to shifting
  • Homogeneous: Only one data type allowed
  • Memory allocation: Needs contiguous block
  • Search complexity: O(n) for unsorted arrays
  • No built-in methods: Manual implementation needed
