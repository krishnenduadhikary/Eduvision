Stacks

A stack is a linear data structure that follows the Last-In-First-Out (LIFO) principle. Elements are added and removed from only one end called the "top" of the stack. This structure is analogous to a stack of plates where you can only take the top plate or add a new plate to the top.

Stacks are fundamental in computer science, used in everything from function call management to expression evaluation and backtracking algorithms.

The last element added to the stack is the first one to be removed. This ordering is called Last-In-First-Out (LIFO).

System Stack

Stacks are used extensively in computer systems:

  • Function call management (call stack)
  • Storing local variables and return addresses
  • Managing execution context

Function A calls B → A pushed to stack

Function B calls C → B pushed to stack

Function C completes → B popped from stack

Function B completes → A popped from stack

struct Node {

struct Node* top = NULL;


void push(int val) {

struct Node* newNode =

newNode->data = val;

newNode->next = top;

top = newNode;

}


int pop() {

if(top == NULL) {

printf("UNDERFLOW");

return -1;

} else {

struct Node* temp = top;

int val = top->data;

top = top->next;

free(temp);

return val;

}

}

Stack Operations

Push Operation

Adds an element to the top of the stack. Must check for overflow in array implementation.

Algorithm
  1. Check if stack is full (TOP == MAX-1)
  2. If full, display OVERFLOW and exit
  3. Increment TOP by 1
  4. Add new element at stack[TOP]

Pop Operation

Removes the top element from the stack. Must check for underflow (empty stack).

Algorithm
  1. Check if stack is empty (TOP == -1)
  2. If empty, display UNDERFLOW and exit
  3. Store value at stack[TOP] in a variable
  4. Decrement TOP by 1

Peek Operation

Returns the top element without removing it. Useful for checking the stack's current state.

int peek() {

if(top == -1) {

printf("STACK IS EMPTY");

return -1;

} else {

return stack[top];

}

}

Other Operations

  • isEmpty: Check if stack is empty (TOP == -1)
  • isFull: Check if stack is full (TOP == MAX-1)
  • Display: Print all stack elements (from top to bottom)
  • Size: Return number of elements (TOP + 1)

Expression Evaluation

Expression Notations

Arithmetic expressions can be written in three different notations:

Infix Notation

Operator between operands

A + B

Human-readable but harder for computers to evaluate

Prefix (Polish)

Operator before operands

+ A B

No parentheses needed, easy to evaluate

Postfix (Reverse Polish)

Operator after operands

A B +

Most efficient for computer evaluation

Operator Precedence and Associativity
Precedence Operator Associativity
Highest (3) Exponentiation (^) Right
Medium (2) Multiplication (*), Division (/) Left
Lowest (1) Addition (+), Subtraction (-) Left

Infix to Postfix Conversion

Algorithm
  1. Push "(" onto stack, add ")" to end of expression
  2. Scan expression left to right:
    • Operand → Add to output
    • "(" → Push to stack
    • Operator → Pop higher precedence operators to output, then push current operator
    • ")" → Pop stack to output until "("
  3. Pop remaining operators from stack
Example Conversion

Infix: A+(B*C-(D/E^F)*G)*H

Postfix: ABC*DEF^/G*-H*+

Prefix Expression Evaluation

Algorithm
  1. Scan expression from right to left
  2. For each element:
    • Operand → Push to stack
    • Operator → Pop two operands, apply operator, push result
  3. Final result is stack top

Example: + - 9 2 7 * 8 / 4 12

Evaluation steps:

1. Push 12, 4 → stack: [12, 4]

2. / → 12/4=3 → stack: [3]

3. Push 8 → stack: [3, 8]

4. * → 3*8=24 → stack: [24]

5. Push 7, 2 → stack: [24, 7, 2]

6. - → 7-2=5 → stack: [24, 5]

7. + → 24+5=29 → stack: [29]

Code Implementation

#include <stdio.h>

int stk[10]; int top=-1;


int main() {

char prefix[10];

int len, val, i, opr1, opr2, res;

printf("\n Enter prefix expression: ");

gets(prefix);

len = strlen(prefix);

for(i=len-1; i>=0; i--) {

if(isdigit(prefix[i])) {

val = prefix[i] - '0';

push(val);

} else {

opr1 = pop(); opr2 = pop();

switch(prefix[i]) {

case '+': res = opr1 + opr2; break;

case '-': res = opr1 - opr2; break;

case '*': res = opr1 * opr2; break;

case '/': res = opr1 / opr2; break;

}

push(res);

}

}

printf("\n RESULT = %d", stk[0]);

return 0;

}

Recursion

Recursion Fundamentals

Recursive Function Properties
  • Base Case: Condition that stops recursion
  • Recursive Case: Function calls itself with modified parameters
  • Progress: Each call moves closer to base case
Example: Factorial

int fact(int n) {

if (n == 0 || n == 1) // Base case

return 1;

else

return n * fact(n-1); // Recursive case

}

fact(5) = 5 * fact(4) = 5 * 24 = 120

fact(4) = 4 * fact(3) = 4 * 6 = 24

fact(3) = 3 * fact(2) = 3 * 2 = 6

fact(2) = 2 * fact(1) = 2 * 1 = 2

fact(1) = 1 (base case)

Memory Allocation

Each recursive call creates a new stack frame:

fact(1)

Stack grows until base case is reached

Types of Recursion
  • Direct: Function calls itself directly
  • Indirect: Function calls another function that calls the original
  • Tail: Recursive call is last operation

Tower of Hanoi

Problem Statement

Move n disks from source pole to destination pole using an auxiliary pole, with rules:

  • Only one disk can be moved at a time
  • No larger disk may be placed on top of a smaller disk
Algorithm
  1. Move n-1 disks from source to auxiliary pole
  2. Move nth disk from source to destination
  3. Move n-1 disks from auxiliary to destination

Code Implementation

void move(int n, char source, char dest, char aux) {

if (n == 1) {

printf("Move disk 1 from %c to %c", source, dest);

} else {

move(n-1, source, aux, dest);

printf("Move disk %d from %c to %c", n, source, dest);

move(n-1, aux, dest, source);

}

}

Fibonacci Sequence

Mathematical Definition

fib(n) = 0 if n == 0

fib(n) = 1 if n == 1 or n == 2

fib(n) = fib(n-1) + fib(n-2) otherwise

Recurrence Relation: T(n) = T(n-1) + T(n-2) + O(1)

Code Implementation

int fib(int n) {

if (n == 0) return 0;

if (n == 1 || n == 2) return 1;

return fib(n-1) + fib(n-2);

}


int main() {

int n = 5;

printf("Fibonacci series: ");

for (int i = 0; i < n; i++) {

printf("%d ", fib(i));

}

return 0;

}

Output: 0 1 1 2 3

Time Complexity: O(2^n) (exponential)

Programming Examples

Array Stack Implementation

#include <stdio.h>

#define MAX 3

int st[MAX], top=-1;


void push(int val) {

if(top == MAX-1) {

printf("\n STACK OVERFLOW");

} else {

top++;

st[top] = val;

}

}


int pop() {

int val;

if(top == -1) {

printf("\n STACK UNDERFLOW");

return -1;

} else {

val = st[top];

top--;

return val;

}

}

This shows the basic array implementation with push and pop operations, including overflow/underflow checks.

Linked Stack Implementation

#include <stdio.h>

#include <stdlib.h>

struct stack {

int data;

struct stack *next;

};

struct stack *top = NULL;


struct stack* push(int val) {

struct stack *ptr;

ptr = (struct stack*)malloc(sizeof(struct stack));

ptr->data = val;

if(top == NULL) {

ptr->next = NULL;

top = ptr;

} else {

ptr->next = top;

top = ptr;

}

return top;

}

This shows the linked list implementation with dynamic memory allocation and pointer manipulation.

Complete Stack Program

#include <stdio.h>

#include <stdlib.h>

#define MAX 3

int st[MAX], top=-1;


void push(int val) { /* ... */ }

int pop() { /* ... */ }

int peek() { return (top == -1) ? -1 : st[top]; }

void display() {

if(top == -1) printf("STACK IS EMPTY");

else for(int i=top; i>=0; i--) printf("%d ", st[i]);

}


int main() {

int val, option;

do {

printf("\n1. PUSH\n2. POP\n3. PEEK\n4. DISPLAY\n5. EXIT");

scanf("%d", &option);

switch(option) {

case 1: /* push */ break;

case 2: /* pop */ break;

case 3: /* peek */ break;

case 4: display(); break;

}

} while(option != 5);

return 0;

}
